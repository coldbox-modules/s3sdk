# Setting up your development environment

- Clone the repository
- `cd test-harness`
- `cp .env.example .env`
- Edit .env file appropriately
- `box install`
- `./start` (Starts server with env vars from .env file)
- Run tests:
  - In browser, visit `http://localhost:60299/tests/runner.cfm`
